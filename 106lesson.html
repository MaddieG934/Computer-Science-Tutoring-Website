Data Structures


Common data structures include arrays, vectors, structures. and classes.

Arrays are lists of a fixed size whose elements are all the same type. They are created like this:

  int myArr = [1, 2, 3];

or like this (for an empty array of size 3):

  myArr(3);

Vectors are also lists whose elements are all the same type, but you can dynamically add and delete element. To use vectors, you need to include the vector library. They are created like this:

  vector<char> myvec = {'a', 'b', 'c'};

or like this (for an empty vector with no capacity):

  vector<char> myvec;

You can add an element to the end of a vector like this:

  myVec.push_back('d');

You can also delete the last element of a vector like this:

  myVec.pop_back(); // push_back() and pop_back() are both "methods" you can use on a vector

Other methods include .size(), which returns the number of elements in a vector; .empty(), which checks whether the vector is empty and returns true or false; and .at(), which takes an element in the vector and returns its position

Structures are containers that can hold elements of multiple types. They are created like this:

  struct {
    int num;
    string myString;
    char myArr(3);
  } structure1, structure2;
  // There will be 2 "instances" of the structure, one named structure1 and the other structure2

You can initialize structure data like this:

  structure1.num = 1;
  structure2.myString = "Hello world";
  // The structure1 variable has its "num" element set to 1
  // The structure2 variable has its "myString" element set to "Hello world"


Classes also can hold elements of different types, as well as functions to manipulaate those elements. They are created like this:

  class Box {
  private: // "private" means only an instance of this class itself can access these variables
    int height, length, width;
  public: // "public" means anyone can access these variables and functions
    // The "constructor" function is called when an instance of the class is first created.
    Box(int h = 0, int l = 0, int w = 0) {
      height = h;
      length = l;
      weight = w;
    }
    double calcVolume() { // This function returns the volume of a particular box by multiplying the three dimensions
      return height * length * width;
    }
  }

You can create an instance of a class like this:

  Box myBox(1, 2, 3); // Create a Box with height 1, length 2, and width 3

You can call public functions like this:

  double volume = myBox.calcVolume();
  // volume = myBox.height * myBox.length * myBox.width = 1 * 2 * 3 = 6

But if you try to access private variables directly like this:

  volume = myBox.height * myBox.length * myBox.width;

you will get an error.
