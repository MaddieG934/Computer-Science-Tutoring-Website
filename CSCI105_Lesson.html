CSCI 105 - Introduction to Computer Science



What is C++?

C++ is a high-performance programming language developed by Bjarne Stroustrup in 1983. C++ is widely used for game development, system software, and real-time applications.



Basic Structure of a C++ Program

A simple C++ program consists of:

  • Preprocessor directives (#include <iostream>)
  • The main function (int main() { })
  • Statements ending with a semicolon (;)
  • Comments (single-line using // and multi-line using /* */)

#include <iostream>
using namespace std;

int main() 
{
    cout << "Hello, World!" << endl;
    return 0;
}


Understanding the Structure

Understanding the structure of a C++ program is essential for beginners because it provides a foundation for writing efficient and error-free code. Knowing how different components such as headers, functions, and statements interact helps in debugging and structuring programs logically. Mastering the basic structure allows for easier learning of more complex concepts in C++.



1 Input and Output

a. Output with “cout”

cout (character output) is used to display data on the screen. The << operator is used to send data to the output stream.

#include <iostream>
using namespace std;

int main() 
{
    cout << "Welcome to C++ programming!";
    return 0;
}


b. Input with "cin"

cin (character input) is used to receive input from the user. The >> operator is used to extract data from the input stream. 

#include <iostream>
using namespace std;

int main()
{
    int age;
    cout << "Enter your age: ";
    cin >> age;
    cout << "You entered: " << age;
    return 0;
}


2 Variables and Data Types

a. Variable Declaration

A variable must be declared before it can be used:

int number;

char letter;
bool isStudent;


b. Common Data Types


            

            

            Data Type
            Description
            

            

            int
            Stores whole numbers (e.g., 10, -5, 100)
            

            

            float
            Stores decimal numbers with single precision (e.g., 10.5f)
            

            

            double
            Stores decimal numbers with double precision (e.g., 10.55)

            bool
            Stores true or false values

3 Control Structures

Control structures are fundamental concepts in C++ programming that determine the flow of execution in a program. They help programmers make decisions, repeat actions, and control the sequence in which code is executed.



Conditional Statements (if, else, switch)

a. if Statement

int age = 18;
if (age >= 18)
{
    cout << "You are eligible to vote.";
}


b.if-else Statement 

 int num = 10;
            if (num > 0)
            {
            cout << "Positive number";
            }
            else
            {
            cout << "Negative number";
            }


c.switch Statement 

 int choice = 2;
            switch (choice)
            {
            case 1:
            cout << "Option 1 selected.";
            break;
            case 2:
            cout << "Option 2 selected.";
            break;
            default:
            cout << "Invalid choice.";
            }


4 Looping Statements (for, while, do-while)

Loops allow a program to repeat a block of code multiple times.



a. for Loop 

 for (int i = 1; i <= 5; i++)
            {
            cout << i << " ";
            }


b. while Loop 

 int i = 1;
 while (i <= 5)
 {
 cout << i << " ";
 i++;
 }


c. do-while Loop 

int i = 1;
do
 {
  cout << i << " ";
  i++;
} while (i <= 5);

6 Arrays and Strings

An array is a fixed-size collection of elements that are all the same data type. Each element is stored in a specific position called an index (starting at 0). Strings are used to store and work with text. To use strings in C++, you must include the library.



a. Arrays 

int numbers[5] = {1, 2, 3, 4, 5};
cout << numbers[0]; // Output: 1


b. Strings 

#include <iostream>
#include <string>
using namespace std;

int main() 
{
    string name = "Alice";
    cout << "Hello, " << name;
    return 0;
}
