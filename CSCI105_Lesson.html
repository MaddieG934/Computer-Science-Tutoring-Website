CSCI 105 - Introduction to C++



What is C++?

C++ is a high-performance programming language developed by Bjarne Stroustrup in 1983. It extends the C language with additional features, including object-oriented programming. C++ is widely used for game development, system software, and real-time applications.



Basic Structure of a C++ Program

A simple C++ program consists of:

  • Preprocessor directives (#include <iostream>)
  • The main function (int main() { })
  • Statements ending with a semicolon (;)
  • Comments (single-line using // and multi-line using /* */)

#include <iostream>
using namespace std;

int main() 
{
    cout << "Hello, World!" << endl;
    return 0;
}


Understanding the Structure

Understanding the structure of a C++ program is essential for beginners because it provides a foundation for writing efficient and error-free code. Knowing how different components such as headers, functions, and statements interact helps in debugging and structuring programs logically. Mastering the basic structure allows for easier learning of more complex concepts in C++.



1 Input and Output

a. Output with “cout”

cout (character output) is used to display data on the screen. The << operator is used to send data to the output stream.

#include <iostream>
using namespace std;

int main() 
{
    cout << "Welcome to C++ programming!";
    return 0;
}


b. Input with "cin"

cin (character input) is used to receive input from the user. The >> operator is used to extract data from the input stream. 

#include <iostream>
using namespace std;

int main()
{
    int age;
    cout << "Enter your age: ";
    cin >> age;
    cout << "You entered: " << age;
    return 0;
}


2 Variables and Data Types

a. Variable Declaration

A variable must be declared before it can be used:

int number;
double price;
char letter;
bool isStudent;


b. Common Data Types


            

            

            Data Type
            Description
            

            

            int
            Stores whole numbers (e.g., 10, -5, 100)
            

            

            float
            Stores decimal numbers with single precision (e.g., 10.5f)
            

            

            double
            Stores decimal numbers with double precision (e.g., 10.55)
            

            

            char
            Stores single characters (e.g., 'A')
            

            

            bool
            Stores true or false values
            

            

            string
            Stores text (requires #include <string>)
